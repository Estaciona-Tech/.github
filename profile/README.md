# 🅿️ Estaciona Tech
> Transformando a experiência de estacionamento com visão computacional, navegação 3D e guia hands-free em tempo real.

Confira nosso site!: https://estaciona.tech/

---

## O que a gente faz

Câmeras fixas detectam a ocupação das vagas automaticamente. Um servidor central processa tudo e serve um app mobile/web com visualização 3D ao vivo. Você verifica disponibilidade antes mesmo de chegar, o sistema te reconhece na entrada e te guia até ela - sem precisar tocar no celular!

---

## Projeto

### [`estaciona-ai`](https://github.com/estaciona-tech/estaciona-ai)

Sistema de guia de estacionamento universitário com detecção em tempo real.

| Camada | Tecnologia |
|--------|-----------|
| Edge (detecção) | Python · YOLOv8n · OpenCV · Sigle-Board Computer |
| Servidor | Rust/Axum + Tokio |
| App | React Native · Three.js |
| Infra | Docker · PostgreSQL · WebSocket · GitHub Actions |
